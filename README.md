# NSS Configuration Files

These files are used by our [mobile application](https://github.com/nss-mitblr/app).

## Office Bearers Format

```json
{
  "programme_officers": [
    {
      "name": "string",
      "contact": "string",
      "email": "string",
    },
    {
      "...": "..."
    }
  ],
  "unit_heads": [
    {
      "name": "string",
      "contact": "string",
      "email": "string",
    },
    {
      "...": "..."
    }
  ]
}
```

## Contact Details Format

```json
{
  "email": "string"
}
```
